# Trash.pl

A simple perl commandline tool for Freedesktop.org-compliant trash management.

## Usage:

```sh
# Send a file to the trash:
trash foo.txt

# List trash contents:
trash -l

# Untrash a file:
trash -u foo.txt

# Empty the trash:
trash -e
```
